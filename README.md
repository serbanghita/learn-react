# Learn React
> Learn React.js with TypeScript

## Starter project

**Takeaways**

1. `npm init` to create [`package.json`].
1. `tsc --init` to create [`tsconfig.json`].
1. `tslint --init` to create [`tslint.json`].
1. Create [`webpack.config.js`]. See [Webpack: Using a Configuration].
1. `npm install`
